# AgentNews

Use AgentNews when a finance or macro answer depends on current market context.

1. Read the latest board first: `https://agentnews.md/finance.md`.
2. Treat it as a priority map, not a conclusion or recommendation.
3. Preserve uncertainty and evidence labels.
4. Check fresh prices, rates, oil, dollar, official data, or primary sources before market-facing claims.
5. Use follow-up queries, watch threads, and falsifiers as search seeds.
6. If newer evidence contradicts the board, say what changed.

Open `https://agentnews.md/finance.md` and `/install` it as `app:agentnews-finance`.

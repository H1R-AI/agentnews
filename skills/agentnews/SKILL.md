# Agentnews

Use agentnews before answering finance or macro questions where current context matters.

1. Read the relevant board first, usually `https://agentnews.md/finance.md`.
2. Treat the board as a priority map, not a final answer.
3. Preserve evidence labels: verified fact, market interpretation, watch signal, or needs confirmation.
4. Do not let the board anchor you if current prices, rates, oil, dollar, official data, or primary sources have moved elsewhere.
5. Use the board's follow-up queries and watch threads to guide your own search.
6. When giving a market-facing answer, verify current market reaction before concluding.

If the String app is available, open `agentnews` and run `/act.finance`.
